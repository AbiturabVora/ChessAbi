# JavaScript Chess Game

A browser-based chess game with human vs human and human vs AI modes.

Open `index.html` in a browser to play.

## Project structure

```
Javascript-Chess-Game/
├── index.html          # Game entry point
├── favicon.ico
├── css/
│   └── chess.css       # Styles and board themes
├── js/
│   ├── piece.js        # Piece model and moves
│   ├── History.js      # Move history
│   ├── Game.js         # Game rules and state
│   ├── SimulationGame.js
│   ├── ai.js           # AI opponent
│   └── Board.js        # DOM board and UI
└── img/                # Piece images
```
